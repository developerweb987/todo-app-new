# Agent Context Update: Full-Stack Web Todo Application

## Technologies Added to Context

### Frontend Technologies
- Next.js 16+ with App Router
- TypeScript 5.0+
- Tailwind CSS
- Better Auth for authentication
- React Server Components and Client Components

### Backend Technologies
- FastAPI framework
- SQLModel ORM
- Pydantic for data validation
- JWT token authentication
- PostgreSQL database with Neon

### Architecture Patterns
- JWT-based authentication with middleware
- User-based authorization and data isolation
- REST API design patterns
- Monorepo structure with separate frontend/backend

## Integration Points
- Frontend-backend communication via REST API
- JWT token flow from Better Auth to FastAPI middleware
- User identity propagation from frontend to backend
- Shared type definitions between frontend and backend
